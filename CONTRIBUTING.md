# Contributing

Thanks for your interest in Pokerai API.

- **API usage / how-to** — check the [docs](https://pokerai.bet/docs.en) and [interactive reference](https://pokerai.bet/reference) first, then open a **Question** issue.
- **Bugs** — open a **Bug** issue and include the **`trace_id`** from the API response; it lets us trace your request end-to-end across services.
- **Feature requests** — open a **Feature** issue describing the use case.
- **Pull requests** — welcome on the public example and SDK repos (see each repo's README). Unless a repo states otherwise, code is MIT and docs are CC BY 4.0.

Real-money-table assistance (RTA) is out of scope and prohibited by our [terms](https://pokerai.bet/terms).
