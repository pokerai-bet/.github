# Security Policy

We take the security of Pokerai API seriously.

## Reporting a vulnerability

**Please do not open a public issue for security problems.**

Report privately via **[GitHub private vulnerability reporting](https://docs.github.com/en/code-security/security-advisories/guidance-on-reporting-and-writing/privately-reporting-a-security-vulnerability)** — go to the **Security** tab of the relevant repository and choose *Report a vulnerability*.

Please include:
- what you found and its impact,
- clear steps to reproduce,
- any relevant `trace_id` from an API response.

We aim to acknowledge within a few business days.

## Good-faith research

Please avoid data destruction, privacy violations, and service disruption. Do not use any finding to assist play at a real-money table — real-time assistance (RTA) is prohibited by our [terms](https://pokerai.bet/terms).
